## Personalizzazione

### Modificare i Contenuti

- Apri `index.html` e modifica i testi nelle varie sezioni
- Aggiorna le informazioni di contatto nella sezione "Contatti"
- Modifica le statistiche nella sezione "Chi Siamo"

### Modificare i Colori

- Apri `styles.css` e modifica le variabili CSS nella sezione `:root`
- I colori principali sono definiti come:
  - `--primary-color`: Colore principale
  - `--secondary-color`: Colore secondario
  - `--accent-color`: Colore di accento

### Modificare le Animazioni

- Le animazioni sono definite in `styles.css` con `@keyframes`
- Le animazioni JavaScript sono gestite in `script.js`
